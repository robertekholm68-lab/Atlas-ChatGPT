# Fas 1 – Appskal och designsystem

Fas 1 etablerar ATLAS visuella och tekniska frontendgrund.

## Levererat

- React 19 + Vite
- Responsivt appskal för mobil, surfplatta och desktop
- Mörkt ATLAS-tema med design tokens
- Desktop-sidebar och mobil bottennavigation
- Fem huvudvyer: Idag, Träning, Kropp, Kost och Coach
- Återanvändbara kort, mätvärden, progressindikatorer och knappar
- Responsiv dashboard-grid
- Interaktiv prototypnavigation
- Interaktiv muskelstatus
- Demosvar och toast-feedback
- Tillgängliga fokusmarkeringar och semantiska navigationsetiketter

## Lokal körning

```bash
npm install
npm run dev
```

## Produktionsbygge

```bash
npm run build
npm run preview
```

## Avgränsning

Fas 1 använder demonstrationsdata. Backend, autentisering och beständig datalagring kopplas in i senare faser.
