# ATLAS assets

Replace these intentional placeholders with final branded production assets without rewriting components.

Recommended structure:

- `program-covers/` — program or workout cover imagery.
- `exercises/` — exercise demonstration images.
- `muscle-figures/` — front/back body figures used by cards and recovery views.
- `muscle-highlights/` — transparent overlays for highlighted target muscles.
- `coach/` — coach avatar/illustration imagery.
- `food/` — meal and ingredient imagery.
- `progress/` — branded progress and achievement graphics.

Asset components should always provide a real `src`, fixed aspect ratio, `object-fit`, accessible `alt`, and an intentional fallback state when a file is missing.
